# `/sandbox` — Micro-app Sandbox (rutas varias)

**Clase:** NATIVO-CANDIDATE (AdSense web permitido en rutas bajo este prefijo).

- `sandbox/page.tsx`: demo/legacy con enlaces rápidos.
- `sandbox/login`: captación + OAuth.
- `sandbox/app/`: Command Hub logueado (PWA `manifest` en layout).
- Rutas `board/` bajo `sandbox/app` reexportan la pizarra principal.

Ver también `sandbox-portal` (un nivel arriba en `src/app/sandbox-portal`).
